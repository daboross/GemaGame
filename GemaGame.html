This is my Java Applet Game, GemaGame.
If it does not load, please update to the Latest Java Version
This game really has no point yet. I currently am just making the physics engine.
The source code is available here: https://github.com/daboross/GemaGame
I am actively coding this game, expect bugs.
Controls:
You can use w Key to jump
You can use a Key to move left
You can use d Key to move right
You can use Left Arrow Key to shoot left
You can use Right Arrow Key to shoot Right
You can use Up Arrow Key to shoot Up
You can use Down Arrow Key to shoot Down
